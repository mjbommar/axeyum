# P2.5 — NRA: incremental linearization → nlsat/CAD

**Size:** XL · **Depends on:** — (independent; the hardest completeness gap) ·
**Frontier**

## Goal
Move nonlinear arithmetic from sound-incomplete (linear abstraction + McCormick)
toward completeness on NRA: first a richer incremental-linearization loop, then a
real CAD/MCSAT oracle for the decidable real fragment.

## Why / leverage
axeyum's NRA/NIA decide-rate trails Z3 substantially. CAD is genuinely hard and
multi-month; the incremental-linearization upgrades are a cheaper middle step that
closes much of the practical gap before committing to CAD.

## Phase exit criteria
- Incremental-linearization upgrades (emonics + richer lemmas + Gröbner) measurably
  raise the NRA/NIA decide-rate (measured), models replay-checked, relaxation-unsat
  certificates retained.
- (Stretch) a CAD/MCSAT path decides the real NRA fragment completely, behind a
  resource budget, with `unknown` first-class outside it.
