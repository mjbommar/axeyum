# P2.1 — BV: lazy blasting, word-level slicing, real theory-checker

**Size:** M · **Depends on:** [P1.4 e-graph](../track-1-engine/P1.4-egraph.md),
[P1.5 CDCL(T)](../track-1-engine/P1.5-cdcl-t-loop.md)

## Goal
Upgrade axeyum's already-strong eager QF_BV to Z3's lazy behavior on the hard ops,
add word-level slicing congruence, and implement real BV theory-checker rules.

## Phase exit criteria
- Lazy multiplier blasting + cheap axioms land and measurably help mul/div-heavy
  QF_BV (measured on the slice), with models still replay-checked.
- BV theory-checker rules are implemented for real (not stubs) and re-validate
  per-bit lemmas.
