# P2.6 — Quantifiers (e-matching, triggers, MBQI, QE)

**Size:** L–XL · **Depends on:** [P1.4 e-graph](../track-1-engine/P1.4-egraph.md),
[P1.5 CDCL(T)](../track-1-engine/P1.5-cdcl-t-loop.md) · **Frontier**

## Goal
Bring quantifier reasoning from finite-domain + first-slice E-matching/MBQI to
production: a Matching Abstract Machine for incremental e-matching, real trigger
inference, lazy clause-form instantiation, MBQI completeness machinery, and
model-based projection for QE.
