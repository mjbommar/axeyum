# P2.3 — EUF on the e-graph (from Ackermann to incremental)

**Size:** M · **Depends on:** [P1.4 e-graph](../track-1-engine/P1.4-egraph.md),
[P1.5 CDCL(T)](../track-1-engine/P1.5-cdcl-t-loop.md) · **First theory on the
loop** (validates the keystones)

## Goal
Decide QF_UF via incremental congruence closure on the e-graph instead of eager
Ackermann reduction. This is the smallest, cleanest first `TheorySolver` and the
proof that the e-graph + CDCL(T) loop work end to end.
