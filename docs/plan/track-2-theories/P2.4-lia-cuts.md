# P2.4 — LIA cut portfolio (native integer solver)

**Size:** M–L · **Depends on:** — (independent; builds on the existing exact
simplex) · **Highest-value single-theory algorithmic upgrade**

## Goal
Replace bit-blast + branch-and-bound LIA with a native integer solver on top of
axeyum's existing exact-rational simplex, adding Z3's cut portfolio. The simplex
already exists, so a native int solver is more attractive than bit-blasting
integers.
