# P2.7 — Strings (unbounded, full `str.*`, regex)

**Size:** L · **Depends on:** [P1.6 theory combination](../track-1-engine/P1.6-theory-combination.md)
(BV+LIA), helps from [P2.6 quantifiers](P2.6-quantifiers.md) · **Frontier**

## Goal
Grow strings from the current bounded, BV-lowered ≤16-byte fragment (ADR-0025/0029,
equality + predicates + length) to QF_S coverage: variable concat/substr/replace,
symbolic indices, regex membership, and reliable `str.len` unsat.

## Why / leverage
Strings are `experimental`: no IR sort, no variable-length results, and `str.len`
unsat can be `unknown` because of the BV+LIA combination gap. Closing that gap is
[P1.6](../track-1-engine/P1.6-theory-combination.md); the rest needs string
*results* (the `Parsed = Term | Str` refactor or a shared `BoundedString` crate).
