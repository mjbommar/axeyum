# P2.2 — Arrays: lazy axiom instantiation

**Size:** L · **Depends on:** [P1.4 e-graph](../track-1-engine/P1.4-egraph.md),
[P1.5 CDCL(T)](../track-1-engine/P1.5-cdcl-t-loop.md) · **The biggest array
scalability win**

## Goal
Move arrays from eager elimination (read-over-write + Ackermann, ADR-0010 — which
blows up on store chains and large index spaces) to **lazy axiom instantiation**
on the e-graph, matching Z3.

## Phase exit criteria
- Lazy arrays decide store-chain / large-index instances that the eager path
  cannot within budget, with replay-checked models; warm path no longer refuses
  arrays (ties into [P4.1](../track-4-usecases-frontend/P4.1-warm-lazy-memory.md)).
- An [Alethe reduction proof](../track-3-proof-lean/P3.5-reduction-proofs.md) for
  read-over-write (`ARRAYS_ROW/IDX/EXT`) becomes feasible (arrays are cvc5's
  fully-checked case).

## Current Boundary (2026-07-10)

ADR-0071 lands abstraction-only base selects and candidate-batched select
congruence. ADR-0072 moves store reasoning out of eager preprocessing on the
canonical ABV/AUFBV route: the shared ROW abstraction records store sites, and
only candidate-violated hit/miss semantics materialize exact guarded axioms.
Function applications in ROW metadata remain shared, projection is function
then array, and original replay still gates SAT. A 24-write concrete-miss chain
replays in one round with zero ROW axioms; hit/miss controls each require one.

The next work is the rest of T2.2.1 plus T2.2.4's non-symbol ownership:
parent-select/ROW/default scheduling on e-graph merges and warm class models for
store/ITE/array-valued-UF terms. ADR-0072 is candidate-guided lazy ROW, not the
complete Z3-style array-theory queue.

ADR-0073 adds the first canonical extensionality slice. Shared `RowCtx` equality
flags receive one diff witness plus paired reads at already-relevant query/store
indices. Only candidate-violated true-congruence or false-diff implications are
materialized, and UF-bearing observation roots remain on the same bus. ADR-0076
expands the matrix to 456/768 equality-bearing comparisons, all clean. This
completes T2.2.3 only for bounded candidate-guided observations: live class-merge
triggers, parent/alternate-path scheduling, e-graph-class models, warm reuse, and
proof artifacts remain.

ADR-0074 adds the first scalable model slice. The shared projection helper now
votes over distinct observed indices, chooses a deterministic majority else
value, and retains only true overrides for both compact BV and generic arrays.
An end-to-end 16-read canonical model uses one round, default `7`, and four
overrides for 12 repeated and four minority values. All differential/public
decisions and replay remain unchanged. T2.2.4 still retains e-graph-class model
ownership, warm reuse, nested/extended arrays, and proof integration.

ADR-0075 closes one proof-side leaf without changing solver behavior: direct
`a=b` plus same-index unequal reads now emits one literal-`select` artifact
accepted in-tree, by Carcara, and by real Lean with no array-elimination trust
step. This is ordinary select congruence. ROW instances, the disequality
diff-witness direction, equality-chain portability, and canonical online proof
logging remain P3.5 work alongside this phase's queue/class-model depth.

ADR-0076 added the first explicit queue state and cross-atom scheduling slice.
For each candidate-false equality, deterministic BFS over candidate-true flags
either leaves the obligation delayed or applies only the missing path-edge reads
at that equality's own diff index. The transitive `a=b ∧ b=c ∧ a≠c` gap closes
with two observations; disconnected SAT, store/UF+ROW composition, and the exact
512 cap are pinned. This avoids eager quadratic preparation, but the queue still
lived across rebuilt canonical rounds rather than inside a backtrackable array
theory attached to `CdclT`.

ADR-0077 supersedes that compensating queue. The flag remains the exact
BV/Boolean atom while its aligned EUF term is the original array equality, so
ordinary reflexivity/transitivity/congruence now runs on the live backtrackable
e-graph. The former transitive, self, store/UF, and 512-observation cases refute
in one round with no array axiom. Projection unions candidate-true direct-symbol
classes and builds one shared majority-default model, closing a transitive SAT
replay gap with disjoint observations. The 790-test library and strengthened
768-comparison matrix are clean. Parent-select merge scheduling, non-symbol/warm
class models, and proof artifacts remain.
