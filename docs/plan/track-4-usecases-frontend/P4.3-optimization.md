# P4.3 — Optimization: OMT lexicographic/Pareto + MILP hardening

**Size:** M · **Depends on:** — (independent; builds on existing `optimize.rs`,
`maxsat.rs`, MILP) · **Use case:** constrained program optimization

## Goal
Grow axeyum's optimization from single-objective (`maximize_bv`/`minimize_lia`,
MaxSAT, MILP — currently `experimental`) to Z3-`opt`-class: **lexicographic** and
**Pareto** multi-objective optimization, with hardened MILP and per-step certified
optima.

## Why / leverage
"Constrained program optimization" is a named first-class use case. The pieces
exist (per-step optimum certified by the underlying procedure; path-condition
optimization landed in `SymbolicExecutor`); what's missing is multi-objective
orchestration and robustness.
