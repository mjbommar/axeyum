# P3.3 — Alethe for QF_BV (the first SMT-level proof)

**Size:** M · **Depends on:** [P3.2 Alethe IR](P3.2-alethe-ir.md)

## Goal
Produce a full, externally-checkable Alethe proof for QF_BV `unsat`: per-operator
`bitblast_*` steps, Tseitin CNF rule steps, and a Boolean refutation
(`resolution`/`drup`/`drat`) over the clauses — checked end-to-end by **Carcara**
(Rust) in CI. This is axeyum's first proof above the clausal layer.

## Approach
- Convert the existing `bitblast_miter` machinery into per-lowered-operator
  `bitblast_*` Alethe steps (`(= <op-term> (@bbterm <bit0> ...))`). axeyum already
  computes exactly these circuits, so the Alethe step is a *record* of the
  lowering.
- Emit CNF rule steps for each Tseitin clause (`tseitin_encode` already knows the
  gate type → the matching `and_pos`/`or_pos`/… rule).
- Close with the clausal refutation from [P3.1](P3.1-lrat.md) (`resolution` from
  LRAT hint chains, or `drat`/`drup`).
- For ops Carcara holes (`bvudiv`/`bvurem`/shifts): emit `hole` for the structural
  step and attach axeyum's **miter certificate** as the side justification (axeyum
  covers div/rem where Alethe does not — a place to lead).
