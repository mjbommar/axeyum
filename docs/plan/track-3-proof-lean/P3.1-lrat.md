# P3.1 — LRAT clausal upgrade

**Size:** S–M · **Depends on:** — (touches the CDCL core; coordinate with
[P1.1](../track-1-engine/P1.1-sat-inprocessing.md) which also emits proof steps)

## Goal
Emit **LRAT** (DRAT + explicit hint chains) from axeyum's proof-producing CDCL
core, with an in-tree `check_lrat`. Keep DRAT for `drat-trim` compatibility. LRAT
is simpler/faster to check and is the format Lean SAT importers prefer — the
clausal sub-layer the Alethe refutation will reference.

- The CDCL core emits LRAT with hint chains; `check_lrat` re-validates every
