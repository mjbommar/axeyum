# Checks

## `z4-regular-module`

Expected result: `sat`.

The validator checks that the listed ring, module addition, and scalar
multiplication tables satisfy the finite left-module laws.

## `submodule-span-replay`

Expected result: `sat`.

The validator checks that `{0, 2}` contains zero, is closed under addition and
scalar multiplication, and equals the submodule generated by `2`.

## `module-hom-kernel-image`

Expected result: `sat`.

The validator checks that multiplication by `2` preserves addition and scalar
multiplication, then recomputes its kernel and image.

## `quotient-module-replay`

Expected result: `sat`.

The validator checks that the listed even and odd cosets partition the module,
then verifies quotient addition and scalar multiplication from representatives.

## `bad-submodule-rejected`

Expected result: `unsat`.

The validator rejects `{0, 1}` because scalar closure fails: `2*1 = 2`, and
`2` is not in the subset. The row links that scalar-closure conflict to checked
QF_UF/Alethe evidence.
