# Model

## Ring

The scalar ring is `Z/4Z`:

```text
0, 1, 2, 3
```

Addition and multiplication are computed modulo `4`. The validator checks the
finite ring table before using it as the scalar ring.

## Module

The module carrier is also:

```text
0, 1, 2, 3
```

Module addition is addition modulo `4`. Scalar multiplication is ring
multiplication modulo `4`:

```text
2 * 1 = 2
2 * 2 = 0
3 * 3 = 1
```

The validator checks:

```text
1*m = m
0*m = 0
r*0 = 0
r*(m+n) = r*m + r*n
(r+s)*m = r*m + s*m
(r*s)*m = r*(s*m)
```

for every scalar and module element in the finite tables.

## Submodule And Span

The submodule generated by `2` is:

```text
{0, 2}
```

The validator recomputes every finite scalar multiple and finite sum over
`Z/4Z`.

## Module Homomorphism

Multiplication by `2` is encoded as:

```text
0 -> 0
1 -> 2
2 -> 0
3 -> 2
```

The kernel and image are both:

```text
{0, 2}
```

The validator checks additive preservation and scalar preservation for every
finite scalar and pair of module elements.

## Quotient Module

The quotient by `{0, 2}` has two cosets:

```text
E = {0, 2}
O = {1, 3}
```

The validator checks the listed quotient addition and scalar-action tables by
applying operations to representatives and then locating the resulting coset.

## Bad Submodule Replay And Certificate

For the rejected subset, exact replay computes:

```text
1 in B
2 not in B
2 * 1 = 2
```

Scalar closure of a submodule would require:

```text
in_subset(2 * 1) = present
```

The finite replay row rejects the malformed submodule directly. The separate
`QF_UF` row then isolates the scalar-closure membership contradiction as an
equality proof obligation. The resource regression checks that Axeyum emits
independently rechecked `UnsatAletheProof` evidence with no trusted reduction
step.
