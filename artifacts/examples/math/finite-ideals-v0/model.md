# Model

## Ring

The scalar ring is `Z/6Z`:

```text
0, 1, 2, 3, 4, 5
```

Addition and multiplication are computed modulo `6`. The validator checks the
finite ring table before using it for ideal or quotient claims.

## Ideal

The even residues form the principal ideal generated by `2`:

```text
(2) = {0, 2, 4}
```

The validator checks:

```text
0 in I
a,b in I -> a + b in I
a in I -> -a in I
r in R, a in I -> r*a in I
r in R, a in I -> a*r in I
```

for every finite ring element and ideal element.

## Ring Homomorphism

Reduction modulo `2` is encoded as:

```text
0 -> 0
1 -> 1
2 -> 0
3 -> 1
4 -> 0
5 -> 1
```

The kernel is `{0, 2, 4}` and the image is `{0, 1}`. The validator checks
zero, one, addition, and multiplication preservation before recomputing the
kernel and image.

## Quotient Ring

The quotient by `{0, 2, 4}` has two cosets:

```text
E = {0, 2, 4}
O = {1, 3, 5}
```

The quotient operations are the `Z/2Z` tables:

```text
E + E = E
E + O = O
O + O = E

E * E = E
E * O = E
O * O = O
```

The validator checks quotient addition and multiplication by applying the
operation to representatives and then locating the resulting coset.
